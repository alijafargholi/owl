Installation¶

Owl relies on Google’s Fire library for automatically generating command line interfaces (CLIs) and Watchdog library for monitor file system.

Installing Requirements¶

Install fire using pip like so:

pip install fire

Then install watchdog:

pip install watchdog

Installing Owl¶

There are 2 ways of getting the latest version.

1- Download the latest version <Owl of Owl by clicking here
  • then unzip anywhere you think it’s more convenient to you. I ideally place it some in your PYTHONPATH

2- Simply clone it in a desire location

git clone https://github.com/alijafargholi/owl.git
